# RK3588 视频流 AI 分析项目：从 0 到 1 计划表

创建时间：2026-06-01 10:34 GMT+8

## 0. 项目目标

从 0 到 1 复刻并完善一个基于 RK3588 的多路视频流 AI 分析系统，最终形成一个适合求职面试展示的完整工程项目。

目标能力：

- 多路 RTSP/本地视频输入
- RK MPP 硬解码
- RKNN NPU 推理
- RGA 图像缩放/格式转换
- 多模型检测：人员、安全帽、打电话/玩手机、疲劳等
- 检测结果融合与告警事件生成
- 多路画面拼图
- MPP H.264/H.265 硬编码
- RTMP/RTSP 推流或本地录像
- 配置化、日志、状态监控、断流重连、优雅退出
- Git 版本管理与阶段性提交记录
- README、架构图、演示视频、面试展示材料

- [ ] `src/` 和 `include/` 一一对应，方便面试时讲清模块边界。

- [ ] 每个模块完成后，在 `docs/dev_log.md` 记录验证结果和 commit id。
- [ ] 第三方依赖尽量只放 RK3588 必要头文件和库，避免把无关 build 产物放入仓库。
